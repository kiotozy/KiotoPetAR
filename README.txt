# Kioto Pet AR

Este projeto usa A-Frame + AR.js para mostrar o personagem Kioto em Realidade Aumentada.

## Como usar:
1. Acesse via GitHub Pages: https://SEU_USUARIO.github.io/KiotoPetAR/
2. Aponte a câmera para o marcador `kioto-marker.patt`
3. O personagem aparecerá em AR e dirá "Meu nome é Kioto Pet"
